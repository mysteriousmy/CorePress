* [ ] 管理系统

  * [ ] 系统状态监控
  * [ ] 系统设置管理
  * [ ] 前端主题管理
  * [ ] 系统用户管理
  * [ ] 系统插件管理
  * [ ] 系统权限管理
  * [ ] 博客管理
  * [ ] 文件管理
  * [ ] more...
* [ ] 博客部分
  * [ ] 博客展示
  * [ ] 博客编辑
  * [ ] 用户注册登录

  * [ ] 简易搜索引擎
  * [ ] 评论（内置)

  * [ ] more...
* [ ] more...

### 技术栈（暂定）

* 后端
  * .Net Core (SDK 7.0)
  * Dappr (ORM)
  * 多个关系型数据库支持（MySQL、PostgreSQL、SQLite)
  * Redis (可选支持)
  * Docker (可选支持)
  * Lucene.net (搜索引擎 暂定)
  * more...
* 前端
  * Solid.js + Typescript
  * Vite
  * FluentUI Web Components
  * ApexCharts

预期目标效果：达到能用，有主流博客系统基本功能部分（碰瓷大的就算了）.
